2D to 3D Reconstruction of Engineering Drawings

PyTorch Graph Nueral Nets

Currently working as part of the Computational Engineering & Robotics Laboratory (CERLAB) at Carnegie Mellon University, and sponsored by MISUMI Manufacturing, this project focuses on the process of taking orthogonal views from 2D engineering drawings and converting them into 3D BRep models using machine learning.

The Challenge

Today, the automatic reconstruction of 3D objects from 2D projections remains a fundamental engineering challenge due to the inherent ambiguity within a set of projected views for an object. Tackling this problem with graph representations and machine learning techniques to describe the relationships between the 2D projection space and the 3D space will allow for higher accuracy in the automatic reconstruction process.

The Solution

Graph based machine learning techniques to describe the relationships of parts between the 2D projections spaces and the 3D space.
