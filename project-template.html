Quadruped Robot Simulation

ROS2 Gazebo Python Control Theory

This project focuses on the design and simulation of a quadruped robot capable of traversing uneven terrain. The primary goal was to implement a Model Predictive Control (MPC) algorithm to stabilize the robot's chassis while maintaining a trot gait.

The Challenge

Developing a stable locomotion controller is difficult due to the non-linear dynamics of legged robots. I utilized Gazebo for physics simulation and ROS2 for the communication architecture between the perception and control nodes.

The Solution

The control architecture was split into two levels:

  • High-level Planner: Determines the foot placement and body trajectory.
  • Low-level Controller: Calculates joint torques using inverse dynamics.

The result was a robust simulation where the robot could recover from external pushes and navigate slopes up to 15 degrees.
